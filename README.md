# qilinbro.github.io

Qilin Bro 的个人 GitHub Pages 站点，记录项目作品、编程统计、创意画廊和朋友们的博客链接。

## 页面入口

- [首页](https://qilinbro.github.io/) — 个人介绍、项目入口和联系方式
- [项目展示](https://qilinbro.github.io/projects.html) — 开源项目和代码片段
- [编程统计](https://qilinbro.github.io/dashboard.html) — WakaTime 编程数据与开发习惯
- [创意画廊](https://qilinbro.github.io/gallery/) — 图片与交互展示
- [邻居们](https://qilinbro.github.io/neighborhood/) — 友链页面

## 本地预览

这个站点是纯静态 HTML/CSS/JavaScript，不需要构建步骤。可以在仓库根目录运行：

```bash
python -m http.server 8000
```

然后打开：

```txt
http://localhost:8000/
```

也可以直接使用 VS Code Live Server 预览。

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages
- WakaTime SVG embeds

## 部署

仓库名为 `qilinbro.github.io`，推送到 `main` 后由 GitHub Pages 直接托管。

如果页面没有及时刷新，可以到 GitHub 仓库的 **Settings → Pages** 检查部署状态。
